> A user authentication using json web token is implemented in this project [ExpressJS, mongoDB, prisma, jsonwebtoken, typescript]

## Install

```sh
npm install
```

### Setup .env file

```js
DATABASE_URL = 
PORT = 
JWT_SECERT = 
JWT_EXPIRATION = 
```

### Start the app

```shell
npm start
```

## Available commands

Running commands with npm `npm run [command]`

| command | description                                   |
| :------ | :-------------------------------------------- |
| `dev`   | Starts a development instance of the forntend |
| `start` | Starts a development instance of the backend  |
